## 🚀 Overview
Trendify is a responsive fashion e-commerce frontend application designed to deliver a modern shopping experience.  
Users can browse products, explore categories, view product details, search items, and manage cart interactions with a smooth UI/UX experience.

---

## ✨ Features
- Responsive Design (Mobile / Tablet / Desktop)
- Men's & Women's Categories
- Product Search & Filtering
- Product Details Page
- Shopping Cart Management
- Quick View Modal
- Reusable Component Architecture
- Modern UI/UX Design
