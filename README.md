# Trendify 👕✨

Modern fashion e-commerce web application built with React, Vite, Redux Toolkit, and Tailwind CSS.

## 🚀 Overview

Trendify is a responsive fashion e-commerce front-end application designed to deliver a modern online shopping experience. The platform allows users to browse men's and women's collections, explore product details, search products, and manage shopping cart interactions through an intuitive and scalable interface.

## ✨ Features

* Responsive Design
* Men's & Women's Categories
* Product Search
* Product Filtering & Sorting
* Product Details Page
* Shopping Cart Management
* Quick View Modal
* Reusable Component Architecture
* Modern UI/UX

## 🛠️ Tech Stack

* React 18
* Vite
* Redux Toolkit
* React Router DOM
* Tailwind CSS
* Framer Motion
* Axios
* React Hook Form

## 📦 Installation

```bash
git clone https://github.com/AbdelrahmanDaify/Trendify.git
cd Trendify
npm install
npm start
```
