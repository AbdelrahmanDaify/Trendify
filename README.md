# Trendify 👕✨

Trendify is a modern fashion e-commerce front-end application built with React, Vite, and Tailwind CSS. The platform provides a seamless shopping experience through intuitive navigation, responsive design, product exploration, category-based browsing, and interactive user interfaces.

Designed with scalability and performance in mind, Trendify delivers a clean and engaging shopping experience for both men's and women's fashion collections.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/AbdelrahmanDaify/Trendify.git
```

### Navigate to Project

```bash
cd Trendify
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```

The application will be available at:

```text
http://localhost:5173
```
